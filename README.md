## undertasker

undertasker is a Windows tool to launch software and execute terminal commands.

![undertasker](https://raw.githubusercontent.com/braycarlson/undertasker/slint/assets/undertasker.png)

## Download

- [Installer](https://github.com/braycarlson/undertasker/releases/)

## Getting Started
1. Download and install the undertasker [installer](https://github.com/braycarlson/undertasker/releases/)
2. You can browse for file(s) to add in undertasker or add a path to an executable and/or a terminal command to `command.json`.
3. Run `undertasker.exe`.

## Note
- `command.json` **must** remain in the same directory as `undertasker`.

## Examples of Commands

### **Executable**
`C:\Program Files (x86)\Steam\Steam.exe`

### **Windows Settings**
`start ms-settings:windowsupdate-action`

### **File Explorer**
`C:\Users\Brayden\Documents`

### **Terminal Command**
`rustup update`
